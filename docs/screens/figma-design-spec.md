# Figmaデザイン仕様書

## 1. 概要

本ドキュメントは、FigmaAIで画面デザインを作成するための詳細な仕様を記載します。
各画面のレイアウト、UI要素、スタイルガイドを定義しています。

## 2. デザインシステム

### 2.1 カラーパレット

#### プライマリカラー
- **プライマリ**: `#2563EB` (Blue 600) - メインボタン、リンク、アクセント
- **プライマリホバー**: `#1D4ED8` (Blue 700)
- **プライマリアクティブ**: `#1E40AF` (Blue 800)

#### セカンダリカラー
- **セカンダリ**: `#64748B` (Slate 500) - サブテキスト、アイコン
- **セカンダリホバー**: `#475569` (Slate 600)

#### 状態カラー
- **成功**: `#10B981` (Green 500) - 成功メッセージ、通知
- **エラー**: `#EF4444` (Red 500) - エラーメッセージ、警告
- **警告**: `#F59E0B` (Amber 500) - 警告メッセージ
- **情報**: `#3B82F6` (Blue 500) - 情報メッセージ

#### ニュートラルカラー
- **背景（ライト）**: `#FFFFFF` (White)
- **背景（グレー）**: `#F8FAFC` (Slate 50)
- **背景（ダーク）**: `#1E293B` (Slate 800)
- **テキスト（プライマリ）**: `#0F172A` (Slate 900)
- **テキスト（セカンダリ）**: `#64748B` (Slate 500)
- **ボーダー**: `#E2E8F0` (Slate 200)
- **ボーダー（ホバー）**: `#CBD5E1` (Slate 300)

### 2.2 タイポグラフィ

#### フォントファミリー
- **メインフォント**: Inter または Noto Sans JP（日本語対応）
- **フォールバック**: -apple-system, BlinkMacSystemFont, "Segoe UI", sans-serif

#### フォントサイズ
- **H1（ページタイトル）**: 32px / 2rem, 太字 (700), 行間 1.2
- **H2（セクションタイトル）**: 24px / 1.5rem, 太字 (700), 行間 1.3
- **H3（サブセクション）**: 20px / 1.25rem, 太字 (600), 行間 1.4
- **Body（本文）**: 16px / 1rem, 通常 (400), 行間 1.5
- **Small（補助テキスト）**: 14px / 0.875rem, 通常 (400), 行間 1.5
- **Label（ラベル）**: 14px / 0.875rem, 中太字 (500), 行間 1.4
- **Caption（キャプション）**: 12px / 0.75rem, 通常 (400), 行間 1.4

### 2.3 スペーシング

#### スペーシングスケール（8px基準）
- **XS**: 4px (0.25rem)
- **S**: 8px (0.5rem)
- **M**: 16px (1rem)
- **L**: 24px (1.5rem)
- **XL**: 32px (2rem)
- **2XL**: 48px (3rem)
- **3XL**: 64px (4rem)

### 2.4 ボーダーラディウス

- **Small**: 4px - 小さな要素（バッジ、アイコン）
- **Medium**: 8px - ボタン、入力欄
- **Large**: 12px - カード、モーダル
- **XLarge**: 16px - 大きなコンテナ

### 2.5 シャドウ

- **Small**: `0 1px 2px 0 rgba(0, 0, 0, 0.05)` - カード、ボタン
- **Medium**: `0 4px 6px -1px rgba(0, 0, 0, 0.1), 0 2px 4px -1px rgba(0, 0, 0, 0.06)` - ドロップダウン
- **Large**: `0 10px 15px -3px rgba(0, 0, 0, 0.1), 0 4px 6px -2px rgba(0, 0, 0, 0.05)` - モーダル

### 2.6 ブレークポイント（レスポンシブ）

- **モバイル**: 320px - 640px
- **タブレット**: 641px - 1024px
- **デスクトップ**: 1025px以上

- ラベルフォント: Label（14px, 中太字）

- ラベルマージン: 下 8px

  - フォント: Body（16px）
  - 背景: 白 `#FFFFFF`
  - ホバー時: ボーダー色 `#CBD5E1`

- パスワード表示/非表示ボタン:
  - 位置: 入力欄の右側、絶対配置
  - サイズ: 32px × 32px
  - アイコン: 目アイコン（表示/非表示）
  - 色: セカンダリ `#64748B`
  - ホバー時: `#475569`
- マージン: 下 24px

**ログインボタン**
- テキスト: "ログイン"
- 幅: 100%
- 高さ: 48px
- フォント: Body（16px, 中太字 500）
- 背景: プライマリ `#2563EB`
- テキスト色: 白 `#FFFFFF`
- ボーダーラディウス: 8px
- シャドウ: Small
- ホバー時: 背景 `#1D4ED8`、シャドウ Medium
- アクティブ時: 背景 `#1E40AF`
- 無効時: 背景 `#CBD5E1`、カーソル not-allowed
- マージン: 下 16px（エラー表示エリアがある場合）

**エラーメッセージ表示エリア**
- 背景: `#FEF2F2`（薄い赤）
- ボーダー: 1px solid `#FECACA`
- ボーダーラディウス: 8px
- パディング: 12px 16px
- テキスト: Small（14px）
- テキスト色: エラー `#EF4444`
- アイコン: 警告アイコン（左側、16px × 16px）
- マージン: 下 16px
- 表示条件: ログイン失敗時のみ

**入力検証エラーメッセージ**
- 位置: 各入力欄の下
- テキスト: Small（14px）
- テキスト色: エラー `#EF4444`
- マージン: 上 4px
- 表示条件: 検証エラー時のみ

**ローディングインジケーター**
- タイプ: スピナーまたはスケルトン
- サイズ: 24px × 24px
- 色: プライマリ `#2563EB`
- 位置: ログインボタン内（中央）またはボタンの代わりに表示
- 表示条件: ログイン処理中のみ

#### 状態別の表示

**初期状態**
- すべての入力欄が空
- エラーメッセージ非表示
- ログインボタン有効

**検証エラー状態**
- 該当する入力欄の下にエラーメッセージ表示
- 入力欄のボーダー色をエラー色（`#EF4444`）に変更
- ログインボタンは有効のまま（または無効化）

**ログイン処理中**
- ログインボタンにローディングインジケーター表示
- ログインボタン無効化
- 入力欄無効化（オプション）

**ログイン失敗状態**
- エラーメッセージ表示エリアにエラーメッセージ表示
- パスワード入力欄をクリア
- メールアドレス入力欄は保持
- ログインボタン再度有効化

#### レスポンシブ対応

**モバイル（320px - 640px）**
- コンテナ幅: 100%（左右パディング 16px）
- パディング: 24px
- フォントサイズ: やや縮小（H1: 28px）

**タブレット（641px - 1024px）**
- コンテナ幅: 400px（中央配置）
- パディング: 32px

**デスクトップ（1025px以上）**
- コンテナ幅: 400px（中央配置）
- パディング: 32px
