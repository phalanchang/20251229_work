# nanobananaAI用 画面遷移図生成プロンプト（超シンプル版）

## 1. 全体の画面遷移図

### プロンプト1: 全体の画面遷移図

```
シンプルなフローチャートを作成してください。

アプリケーション → 認証状態チェック → 未ログイン/ログイン済み
未ログイン → ログインページ
ログイン済み → ダッシュボード
ログインページ → 保護されたページ
ダッシュボード → 保護されたページ

白背景、シンプルなボックスと矢印、日本語テキスト。
```

---

## 2. ログインフロー詳細図

### プロンプト2: ログインフロー詳細図

```
ログイン処理のフローチャートを作成してください。

アプリ起動 → 認証状態チェック → 未ログイン/ログイン済み
未ログイン → ログインページ表示
ログイン済み → ダッシュボード
ログインページ → 認証情報入力 → 入力検証 → 検証OK/検証エラー
検証OK → API送信 → 認証処理 → 認証成功/認証失敗
認証成功 → ダッシュボード
認証失敗 → エラー表示

白背景、シンプルなボックスと矢印、日本語テキスト。
```

---

## 3. ログアウトフロー図

### プロンプト3: ログアウトフロー図

```
ログアウト処理のフローチャートを作成してください。

任意のページ → ログアウトボタンクリック → ログアウト確認 → キャンセル/確認
キャンセル → 元のページ
確認 → API送信 → セッション無効化 → トークン削除 → ログインページ

白背景、シンプルなボックスと矢印、日本語テキスト。
```

---

## 4. 保護されたページへのアクセスフロー図

### プロンプト4: 保護されたページアクセスフロー図

```
保護されたページへのアクセスフローチャートを作成してください。

保護されたページのURLにアクセス → 認証状態チェック → 未ログイン/ログイン済み
未ログイン → ログインページにリダイレクト → ユーザーがログイン → ログイン成功 → 元のページにリダイレクト
ログイン済み → ページを表示

白背景、シンプルなボックスと矢印、日本語テキスト。
```

---

## 5. 認証状態遷移図

### プロンプト5: 認証状態遷移図

```
認証状態の遷移図を作成してください。

未ログイン ←→ ログイン済み
未ログイン → ログイン済み（ログイン成功）
ログイン済み → 未ログイン（ログアウト）
ログイン済み → 未ログイン（セッション期限切れ）
ログイン済み → ログイン済み（通常使用、自己ループ）

白背景、シンプルな円形ボックスと矢印、日本語テキスト。
```

---

## 6. 包括的な画面遷移図（推奨）

### プロンプト6: 包括的な画面遷移図

```
ログイン・ログアウト機能の画面遷移図を1枚で作成してください。

アプリケーション → 認証状態チェック → 未ログイン/ログイン済み

未ログイン → ログインページ → 認証情報入力 → 入力検証 → 認証処理 → 認証成功/認証失敗
認証成功 → ダッシュボード
認証失敗 → エラー表示

ログイン済み → ダッシュボード → ログアウト → ログインページ

保護されたページ → 認証状態チェック → 未ログイン/ログイン済み
未ログイン → ログインページ
ログイン済み → ページ表示

白背景、シンプルなボックスと矢印、日本語テキスト。
```

---

## 7. 使用上の注意

### プロンプトの使い方

- **全体像**: プロンプト6を使用
- **特定のフロー**: プロンプト2-5を使用
- **シンプルな全体図**: プロンプト1を使用

### 生成時のコツ

1. プロンプトをそのままコピーして使用
2. 生成後に「色を追加」「矢印を太く」などで調整
3. 複数バージョンを生成して比較

### 調整用プロンプト例

生成後に以下のような指示で調整できます：
- 「色を追加してください（成功=緑、エラー=赤）」
- 「矢印を太くしてください」
- 「よりコンパクトにしてください」
