# プロンプトファイル作成ツール

## 概要

`YYYYMMDD_hhmm.md` 形式のプロンプト用マークダウンファイルを自動生成するバッチファイルです。

## 使い方

### 推奨: PowerShell版

1. `create-prompt-file-ps.bat` をダブルクリック
2. ファイルが自動的に作成されます
3. ファイルを開くかどうか聞かれたら、Y/Nで選択

### 代替: 標準バッチ版

1. `create-prompt-file.bat` をダブルクリック
2. ファイルが自動的に作成されます
3. ファイルを開くかどうか聞かれたら、Y/Nで選択

## ファイル名の形式

- 形式: `YYYYMMDD_hhmm.md`
- 例: `20251230_0922.md`（2025年12月30日 9時22分）

## 注意事項

- ファイルが既に存在する場合は、上書きするかどうか確認されます
- バッチファイルは `docs/prompt` ディレクトリに配置してください
- 作成されるファイルも同じディレクトリに保存されます

## トラブルシューティング

### PowerShell版が動作しない場合

- 実行ポリシーの問題の可能性があります
- その場合は、標準バッチ版（`create-prompt-file.bat`）を使用してください

### ファイルが作成されない場合

- バッチファイルの実行権限を確認してください
- ウイルス対策ソフトがブロックしていないか確認してください
