# 2ndスプリント ドキュメント

## 概要

2ndスプリント用のドキュメントを管理するディレクトリです。

## ディレクトリ構成

```
2nd_sprint/
├── README.md           # このファイル
├── screens/            # 画面設計関連
│   ├── screen-list.md  # 画面一覧
│   └── ...             # その他の画面設計ドキュメント
└── input_items/        # 外部ツール（FigmaAI等）への入力用アイテム
```
